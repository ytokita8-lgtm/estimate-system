# 🏗️ 木造アパート新築工事 見積システム

**創世建設株式会社** 内製ツール

> 住所入力から都市計画情報の自動取得・図面AI解析・見積計算・Excel/PDF出力まで一貫して対応する建築見積システム

---

## 🚀 使い方

**ブラウザで `index.html` を開くだけ**（インストール不要）

または公開URL：`https://[GitHubユーザー名].github.io/estimate-system/`

---

## 📋 機能一覧

| ステップ | 機能 |
|---|---|
| 01 住所・都市計画 | 住所入力 → AIがWeb検索して都市計画を自動取得 |
| 02 図面・写真読込 | ラフ図・手書き・PDF → AIが自動解析・数値入力 |
| 03 仕様・数量設定 | 構造・間取り・面積・建具数量を設定 |
| 04 見積確認 | 全工種の内訳明細を表示・原価管理モード対応 |
| 05 出力・保存 | Excel出力・印刷/PDF・案件データ保存/再読込 |

### 自動設定される仕様
- **防火・準防火地域** → サッシ・ドアを防火仕様単価に自動変更
- **3階建て** → 準防火仕様を自動ON
- **埋蔵文化財包蔵地** → GRRシート工法を自動計上
- **都市ガス/プロパン** → ガス工事単価を自動切替

## ⚙️ 技術仕様

- **フロントエンド**: HTML / CSS / Vanilla JS（フレームワーク不使用）
- **AI解析**: Claude API (claude-sonnet-4-20250514)
- **地図・住所検索**: Nominatim (OpenStreetMap)
- **Excel出力**: VBA互換HTMLテーブル形式
- **依存ライブラリ**: なし（CDN不使用・完全スタンドアロン）

---

## 🔑 API設定について

このシステムはClaude APIを使用します。  
`index.html` を開いた状態で、Claude.aiにログインしていれば自動的に動作します。
